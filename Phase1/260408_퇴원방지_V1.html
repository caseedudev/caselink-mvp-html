01 · OVERVIEW
시스템 개요
강사가 수업일지에 입력하는 출결 · 과제 · WT점수 · 참여도 4개 변수를 1주 단위로 누적 스코어링합니다. 학생별 주간 위험점수가 임계값을 넘으면 주의 → 경고 → 예상퇴원 3단계 알림이 자동 발생합니다. 경고 이상은 부서장에게도 즉시 전달됩니다.
🟡 주의 (CAUTION)
초기 이탈 징후 감지. 담당 강사가 해당 주 안에 개별 면담을 진행합니다.
주간 위험점수 30점 이상
🔴 경고 (WARNING)
복수 징후 동시 발생. 학부모 전화 상담 및 부서장 보고가 필요합니다.
주간 위험점수 60점 이상 또는 즉각 트리거
🚨 예상퇴원 (CRITICAL)
퇴원 결심 임박. 원장·부서장 직접 개입 및 학부모 긴급 연락.

02 · INPUT VARIABLES
입력 변수 정의 및 점수 변환
수업일지 입력 시 아래 5개 변수를 기록합니다. 각 값은 위험 포인트(pts)로 변환되어 주간 합산됩니다.
① 출결 (attendance) — 5단계
입력값위험 포인트설명
출석0 pts정상
지각 (사전 연락)0 pts사전에 연락하고 늦게 입실 — 점수 차감 없음
결석 (사전 연락)0 pts사전에 연락하고 불참 — 점수 차감 없음
무단지각5 pts연락 없이 늦게 입실
무단결석25 pts연락 없이 불참 — 즉각 경고 트리거 병행 발동
주간 누적 가중 규칙 — 같은 주에 무단지각·무단결석이 2회 이상이면 출결 합산 점수를 ×1.5 가중합니다.
예) 무단결석 1회(25) + 무단지각 1회(5) → (25+5) ×1.5 = 45 pts
② 과제 수행도 (homework) — 3단계
입력값위험 포인트설명
우수0 pts완성도 높음
보통8 pts부분 미완성 또는 성의 부족
미흡20 pts미제출·답 베끼기 포함 — 즉각 경고 트리거 병행
연속 미흡 가중 규칙 — 2주 연속 미흡이면 다음 주 과제 점수를 ×1.5 가중합니다.
특이사항란에 "답 베낌" "찍음" 키워드가 감지되면 즉각 경고 트리거를 별도 발동합니다.
③ 주간테스트(WT) — 반 평균 대비 차이 + 추이 기준
판단 기준조건위험 포인트
반 평균 대비 차이
해당 주 1회 적용		 평균 이상0 pts
평균 미만 ~ –9점5 pts
평균 –10점 ~ –19점15 pts
평균 –20점 이상25 pts
점수 추이 (연속 하락)
주간 확정 후 추가 적용		 하락 없음 또는 상승+0 pts
2주 연속 하락+10 pts 추가
3주 이상 연속 하락+20 pts 추가 + 즉각 경고 트리거
반 평균 계산 — 동일 수업 내 같은 주 WT 점수의 산술 평균으로 자동 계산됩니다.
결석 등으로 WT 미응시한 주는 추이 계산에서 제외합니다.
④ 수업 참여도 (participation) — 3단계
입력값위험 포인트설명
우수0 pts적극적 참여, 질문 있음
보통5 pts수동적 참여, 질문 없음
미흡15 pts엎드림·딴짓·시선 회피·대답 거부
⑤ 특이사항 키워드 감지 (remark_keywords)
키워드 / 패턴위험 포인트트리거
"답 베낌" "찍음" "해설지 베낌"+15 pts🔴 즉각 경고
"엎드림" "졸음" "수면"+10 pts—
"질문없음" "시선회피" "대답안함"+8 pts—
"타학원" "레벨테스트" "다른학원"+20 pts🔴 즉각 경고
"독학" "인강" "혼자 공부"+30 pts🚨 즉각 예상퇴원
"왜 공부" "그만하고 싶다" "못하겠다"+30 pts🚨 즉각 예상퇴원
키워드는 특이사항 입력란 문자열 포함 여부로 감지합니다. 강사가 선택 가능한 체크박스 태그 형태로도 제공하면 감지율이 높아집니다.
03 · WEEKLY SCORING ENGINE
주간 위험점수 계산 알고리즘
매주 월요일 자정 직전 7일간의 수업일지 데이터를 집계하여 위험점수를 확정합니다. 재원기간 · 학년 · 성향 보정계수를 곱한 최종 점수로 등급을 결정합니다.
// ══ CaseLink 주간 위험점수 계산 ══ function calcWeeklyRisk(student, weekRecords) { let pts = 0 // ① 출결 합산 (사전연락 지각·결석 0pts, 무단지각·무단결석만 집계, 주간 2회+ 시 ×1.5) const absCount = weekRecords.filter(r => r.attendance === '무단지각' || r.attendance === '무단결석' ).length const attPts = weekRecords.sumBy(attendanceScore) pts += absCount >= 2 ? attPts * 1.5 : attPts // ② 과제 합산 (2주 연속 미흡이면 이번 주 ×1.5) const hwStreak = consecutivePoorHomework(student.id, 2) const hwPts = weekRecords.sumBy(homeworkScore) pts += hwStreak ? hwPts * 1.5 : hwPts // ③ WT — 반 평균 대비 차이 점수 const wtGap = calcWtGap(student, weekRecords) // 내 점수 − 반 평균 pts += wtGapScore(wtGap) // ③-b WT — 연속 하락 추이 가산점 const dropWeeks = consecutiveWtDrop(student.id) if (dropWeeks >= 3) { pts += 20; triggerWarning(student) } else if (dropWeeks === 2) pts += 10 // ④ 참여도 합산 pts += weekRecords.sumBy(participationScore) // ⑤ 키워드 포인트 pts += weekRecords.sumBy(keywordScore) // ⑥ 보정 계수 적용 pts *= enrollmentWeight(student.enrollment_months) pts *= gradeWeight(student.grade_group) if (student.personality === '적응형') pts *= 1.2 return { score: Math.round(pts), grade: toGrade(pts) } } function toGrade(pts) { if (pts >= 90) return '예상퇴원' if (pts >= 60) return '경고' if (pts >= 30) return '주의' return '정상' }
📈 주간 위험점수 → 등급 기준
정상 (0~29)
🟡 주의 (30~59)
🔴 경고 (60~89)
🚨 (90+)
※ 즉각 트리거 발동 시 점수와 무관하게 해당 등급으로 즉시 상향됩니다.
⚙️ 보정 계수
구분조건계수이유
재원기간1개월 미만 초단기×1.5첫 적응 실패 → 즉각 퇴원 직결
1~3개월 단기×1.3과제 누적 스트레스 시기
3개월~1년 중기×1.0표준 기준
1년 이상 장기×0.9매너리즘, 급격한 변화가 오히려 위험신호
학년군초등×1.2부모 의존도 높아 아이 불만 = 즉각 퇴원
중등×1.0표준 기준
고등×0.85번아웃성 결석은 일시적일 수 있음
성향적응형 (조용한 학생)×1.2퇴원생의 70~80% — 겉으로 안 보임
04 · INSTANT TRIGGERS
점수 무관 즉각 트리거
아래 사건은 주간 위험점수와 무관하게 발생 즉시 해당 등급을 강제 발동합니다.
🔴 즉각 경고 트리거
  • 과제 답 베끼기 / 찍기 1회 적발
  • 무단결석 + 당일 학부모 연락 없음
  • WT 점수 3주 연속 하락
  • 타 학원 레벨테스트 정황 포착
  • 단짝 친구 퇴원 완료
  • 레벨 다운(반 하향) 조치 직후
🚨 즉각 예상퇴원 트리거
  • "독학" "인강" "혼자 공부" 키워드 감지
  • "왜 공부해야 하는지" "그만하고 싶다" 발언
  • 무단결석 후 학생 + 학부모 모두 연락 두절
  • 타 학원 레벨테스트 + 무단결석 동시 발생
  • 즉각 경고 트리거 2개 이상 같은 주 발동
05 · GRADE-BASED RULES
학년별 세부 주간 발동 기준
초등 주간 발동 기준
🟡 주의 — 해당 주 중 1개 이상
  • 지각 또는 결석 주 1회
  • 참여도 미흡 주 1회
  • 단짝 친구가 퇴원 조짐을 보이기 시작
🔴 경고 — 해당 주 중 1개 이상
  • 과제 미흡 주 2회 이상
  • 결석 + 과제 미흡 동시
  • 학부모 "과제 너무 많다" 피드백 접수
🚨 예상퇴원 — 1개 이상 즉시
  • 무단결석 주 1회
  • 단짝 친구 퇴원 완료 (동반이탈 최고 위험)
  • 즉각 예상퇴원 트리거 발동
중등 주간 발동 기준
🟡 주의 — 해당 주 중 1개 이상
  • WT 점수 반 평균 –10점 이상
  • 참여도 미흡 주 2회 이상
  • 지각 주 2회
🔴 경고 — 해당 주 중 1개 이상
  • 과제 답 베끼기 1회 적발 (즉각 트리거)
  • WT 2주 연속 하락 + 반 평균 –15점
  • 보강/클리닉 무단 불참 주 1회
  • 레벨 다운 조치 직후 (즉각 트리거)
🚨 예상퇴원 — 1개 이상 즉시
  • 타 학원 레벨테스트 정황 포착 (즉각 트리거)
  • 소통 완전 단절 + 무단결석 동시
  • 경고 등급 2주 이상 지속 + 개선 없음
고등 주간 발동 기준
🟡 주의 — 해당 주 중 1개 이상
  • 수업 중 졸음 주 2회 이상 (번아웃 초기)
  • 내신·모의고사 직후 WT 급락 (전주 대비 –15점+)
🔴 경고 — 해당 주 중 1개 이상
  • 특정 요일 결석 2주 연속 반복 (회피 패턴)
  • WT 3주 연속 하락 (즉각 트리거)
  • "잠 못 잔다" 번아웃 호소 주 2회
🚨 예상퇴원 — 1개 이상 즉시
  • "왜 공부해야 하는지 모르겠다" 동기상실 발언 1회 (즉각 트리거)
  • 무단결석 후 학생 + 학부모 연락 두절 (즉각 트리거)
  • 예체능 전환 언급 + 결석 동시
06 · TENURE-BASED RULES
재원기간별 추가 감시 기준
구간핵심 감시 지표즉각 발동 조건
초단기
1개월 미만		 첫 주 결석, 부모 불만 첫 달 결석 주 1회 이상 → 즉각 경고
부모 "수업방식 안 맞는다" → 즉각 경고
단기
1~3개월		 과제 미완성, 교우 미형성 쉬는 시간 혼자 겉돌기 2주 연속 → 경고
과제 미흡 2주 연속 → 경고
중기
3개월~1년		 시험 결과, 친구 퇴원 분기 고사 후 레벨 다운 → 즉각 경고
단짝 친구 퇴원 완료 → 즉각 경고
장기
1년 이상		 매너리즘 무기력, 사춘기 WT 전주 대비 –15점+ 급락 → 즉각 경고
독학 언급 → 즉각 예상퇴원
07 · PERSONALITY ADJUSTMENT
성향별 징후 해석 및 강사 개입 가이드
퇴원보고서 분석 결과 적응형(조용한 학생)이 70~80%를 차지합니다. 겉으로 이상 없어 보이다가 갑자기 퇴원 통보를 하므로 점수를 ×1.2 증폭합니다.
🟦 적응형 — 조용하지만 고위험
숨겨진 주간 위험신호
  • 참여도 미흡 3주 연속 → 경고
  • 질문 0회 수업 2주 연속
  • 단짝 친구 퇴원 → 즉각 경고
  • "독학" 언급 1회 → 즉각 예상퇴원
🗣️ 강사 개입 — 나 전달법
"OO이가 요즘 조용히 혼자 풀고만 있어서(사실), 혹시 힘든 게 있는 건 아닐까 봐 걱정이 돼(감정). 모르는 거 있으면 언제든 편하게 물어봐 줬으면 해(바람)."
🟩 표현형 — 신호가 명확하게 드러남
노출된 주간 위험신호
  • 불만 표출 + 과제 미흡 → 주의
  • 동아리로 지각·결석 반복 → 경고
  • 교우 마찰 + 감정 폭발 → 경고
  • 무단결석 + 학부모 동의 → 즉각 예상퇴원
🗣️ 강사 개입 — 나 전달법
"최근 지각이 잦고 과제가 밀리는 걸 보니(사실), 진도를 못 따라갈까 봐 조바심이 나(감정). 활동도 좋지만 수업 시간은 지켜줬으면 해(바람)."
08 · ALERT ACTIONS
등급별 시스템 자동 액션
🟡 주의 — 담당 강사 액션
  • 학생 배지 "주의"로 변경
  • 강사 화면 팝업: "이번 주 안에 OO학생과 짧은 1:1 대화를 권장합니다."
  • 수업일지 특이사항 입력 필수 표시 활성화
  • 재원 초단기/단기 학생: "이번 주 안에 학부모 안부 전화" 가이드 표시
🔴 경고 — 강사 + 부서장 액션
  • 상태 배지 "경고" + 강사 화면 상단 고정 배너
  • 부서장 즉시 알림: "⚠️ [학생명] 경고 등급 — 학부모 상담 필요"
  • 상담 관리 탭에 학부모 전화상담 TODO 자동 생성 (기한: 이번 주 내)
  • 카카오 알림톡 전송 버튼 원클릭 활성화
🚨 예상퇴원 — 원장 + 부서장 + 강사 전체 액션
  • 상태 배지 "예상퇴원" + 대시보드 위험군 카드 즉시 추가
  • 원장·부서장 푸시 알림: "🚨 [학생명] 퇴원 위험 — 즉각 개입 필요"
